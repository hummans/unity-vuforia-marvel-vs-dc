## Requirements

* Unity3d 2020.1.5f1
* Visual Studio Code 1.2.3
* UniRx 7.1.0
* ARFoundation 4.0.12
* ARCore XR Plugin 4.0.12
* Vuforia XR Plugin 10.0.12
* TextMeshPro 3.0.6
* Input System 1.0.2
